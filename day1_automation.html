                  + `‡∏à‡∏≥‡∏ô‡∏ß‡∏ô‡∏™‡∏¥‡∏ô‡∏Ñ‡πâ‡∏≤: ${tranQty} ‡∏ä‡∏¥‡πâ‡∏ô\n`
                  + `‡∏£‡∏≤‡∏Ñ‡∏≤‡∏ï‡πà‡∏≠‡∏´‡∏ô‡πà‡∏ß‡∏¢: ${unitPrice} ‡∏ö‡∏≤‡∏ó\n`

/**
 * ‡∏ü‡∏±‡∏á‡∏Å‡πå‡∏ä‡∏±‡∏ô‡∏Ñ‡πâ‡∏ô‡∏´‡∏≤‡∏≠‡∏µ‡πÄ‡∏°‡∏•‡∏•‡∏π‡∏Å‡∏Ñ‡πâ‡∏≤‡∏à‡∏≤‡∏Å CusID ‡πÉ‡∏ô‡∏ä‡∏µ‡∏ï Customer
 * @param {string} customerId ‡∏£‡∏´‡∏±‡∏™‡∏•‡∏π‡∏Å‡∏Ñ‡πâ‡∏≤‡∏ó‡∏µ‡πà‡∏ï‡πâ‡∏≠‡∏á‡∏Å‡∏≤‡∏£‡∏Ñ‡πâ‡∏ô‡∏´‡∏≤
 * @returns {string | null} ‡∏≠‡∏µ‡πÄ‡∏°‡∏•‡∏•‡∏π‡∏Å‡∏Ñ‡πâ‡∏≤ ‡∏´‡∏£‡∏∑‡∏≠ null ‡∏ñ‡πâ‡∏≤‡πÑ‡∏°‡πà‡∏û‡∏ö
 */
function getCustomerEmail(customerId) {
    
    try {
        const ss = SpreadsheetApp.openById(SPREADSHEET_ID);
        const sheet = ss.getSheetByName(CUSTOMER_SHEET_NAME);
        
        if (!sheet) {
            Logger.log(`ERROR: Customer Sheet not found with name: ${CUSTOMER_SHEET_NAME}`);
            return null;
        }

        const lastRow = sheet.getLastRow();
        if (lastRow < FIRST_DATA_ROW) return null; // ‡πÑ‡∏°‡πà‡∏°‡∏µ‡∏Ç‡πâ‡∏≠‡∏°‡∏π‡∏•
        
        // ‡∏î‡∏∂‡∏á‡∏Ç‡πâ‡∏≠‡∏°‡∏π‡∏•‡∏ï‡∏±‡πâ‡∏á‡πÅ‡∏ï‡πà‡πÅ‡∏ñ‡∏ß‡∏ó‡∏µ‡πà 2 (FIRST_DATA_ROW) ‡∏ñ‡∏∂‡∏á‡∏Ñ‡∏≠‡∏•‡∏±‡∏°‡∏ô‡πå CusEmail (F)
        const numColsToFetch = EMAIL_COLUMN_CUS - ID_COLUMN_CUS + 1;
        const dataRange = sheet.getRange(FIRST_DATA_ROW, ID_COLUMN_CUS, lastRow - (FIRST_DATA_ROW - 1), numColsToFetch);
        const values = dataRange.getValues();

        // ‡∏Ñ‡πâ‡∏ô‡∏´‡∏≤‡∏≠‡∏µ‡πÄ‡∏°‡∏•‡∏à‡∏≤‡∏Å CusID
        for (let i = 0; i < values.length; i++) {
            const rowData = values[i];
            const currentCusId = rowData[0]; // CusID ‡∏≠‡∏¢‡∏π‡πà‡∏ó‡∏µ‡πà Index 0 (‡∏Ñ‡∏≠‡∏•‡∏±‡∏°‡∏ô‡πå A)
            const email = rowData[EMAIL_COLUMN_CUS - ID_COLUMN_CUS]; // Index 5 ‡∏™‡∏≥‡∏´‡∏£‡∏±‡∏ö‡∏Ñ‡∏≠‡∏•‡∏±‡∏°‡∏ô‡πå F (CusEmail)

            // üí° ‡∏™‡∏≥‡∏Ñ‡∏±‡∏ç: ‡πÉ‡∏ä‡πâ trim() ‡πÄ‡∏û‡∏∑‡πà‡∏≠‡∏Å‡∏≥‡∏à‡∏±‡∏î‡∏ä‡πà‡∏≠‡∏á‡∏ß‡πà‡∏≤‡∏á‡∏ó‡∏µ‡πà‡∏≠‡∏≤‡∏à‡∏ó‡∏≥‡πÉ‡∏´‡πâ‡∏Å‡∏≤‡∏£‡πÄ‡∏õ‡∏£‡∏µ‡∏¢‡∏ö‡πÄ‡∏ó‡∏µ‡∏¢‡∏ö‡∏•‡πâ‡∏°‡πÄ‡∏´‡∏•‡∏ß
            if (currentCusId && currentCusId.toString().trim() === customerId.toString().trim()) {
                // üí° ‡πÉ‡∏ä‡πâ trim() ‡∏Å‡πà‡∏≠‡∏ô return ‡πÄ‡∏û‡∏∑‡πà‡∏≠‡∏õ‡πâ‡∏≠‡∏á‡∏Å‡∏±‡∏ô‡∏ä‡πà‡∏≠‡∏á‡∏ß‡πà‡∏≤‡∏á‡πÉ‡∏ô‡∏≠‡∏µ‡πÄ‡∏°‡∏•
                return email.toString().trim(); 
            }
        }
        
        return null;

    } catch (error) {
        Logger.log(`ERROR in getCustomerEmail: ${error.toString()}`);
        return null;
    }
}

/**
 * ‡∏™‡∏Ñ‡∏£‡∏¥‡∏õ‡∏ï‡πå‡∏ô‡∏µ‡πâ‡∏à‡∏∞‡∏™‡∏£‡πâ‡∏≤‡∏á ID ‡∏•‡∏π‡∏Å‡∏Ñ‡πâ‡∏≤‡∏ó‡∏µ‡πà‡πÑ‡∏°‡πà‡∏ã‡πâ‡∏≥‡∏Å‡∏±‡∏ô (Primary Key) ‡πÇ‡∏î‡∏¢‡∏≠‡∏±‡∏ï‡πÇ‡∏ô‡∏°‡∏±‡∏ï‡∏¥

function onEdit(e) {
  
  // üí° ‡πÅ‡∏Å‡πâ‡πÑ‡∏Ç: ‡πÉ‡∏ä‡πâ TARGET_SHEET_NAME ‡∏à‡∏≤‡∏Å Configs ‡πÄ‡∏û‡∏∑‡πà‡∏≠‡∏Ñ‡∏ß‡∏≤‡∏°‡∏™‡∏≠‡∏î‡∏Ñ‡∏•‡πâ‡∏≠‡∏á
  const TARGET_SHEET_NAME = CUSTOMER_SHEET_NAME; 
  // üí° ‡πÅ‡∏Å‡πâ‡πÑ‡∏Ç: ‡∏Ñ‡∏≠‡∏•‡∏±‡∏°‡∏ô‡πå A ‡∏Ñ‡∏∑‡∏≠‡∏Ñ‡∏≠‡∏•‡∏±‡∏°‡∏ô‡πå‡∏ó‡∏µ‡πà‡∏à‡∏∞‡πÉ‡∏™‡πà ID (Index 1)
  const ID_TARGET_COLUMN = ID_COLUMN_CUS; 

  if (!e || !e.range) return;

  const range = e.range;
  const sheet = range.getSheet();
  
  if (sheet.getName() !== TARGET_SHEET_NAME) return;

  const row = range.getRow();
  const column = range.getColumn();

  // 1. ‡∏ï‡∏£‡∏ß‡∏à‡∏™‡∏≠‡∏ö‡πÅ‡∏ñ‡∏ß: ‡∏ï‡πâ‡∏≠‡∏á‡πÑ‡∏°‡πà‡πÄ‡∏õ‡πá‡∏ô‡πÅ‡∏ñ‡∏ß‡∏™‡πà‡∏ß‡∏ô‡∏´‡∏±‡∏ß
  if (row < FIRST_DATA_ROW) return;

  // 2. ‡∏ï‡∏£‡∏ß‡∏à‡∏™‡∏≠‡∏ö‡∏ß‡πà‡∏≤‡πÄ‡∏õ‡πá‡∏ô‡∏Å‡∏≤‡∏£‡πÅ‡∏Å‡πâ‡πÑ‡∏Ç‡πÉ‡∏ô‡∏Ñ‡∏≠‡∏•‡∏±‡∏°‡∏ô‡πå ID (A) ‡∏´‡∏£‡∏∑‡∏≠‡πÑ‡∏°‡πà
  if (column === ID_TARGET_COLUMN) return; // ‡∏´‡∏≤‡∏Å‡πÅ‡∏Å‡πâ‡πÑ‡∏Ç‡∏Ñ‡∏≠‡∏•‡∏±‡∏°‡∏ô‡πå ID ‡πÇ‡∏î‡∏¢‡∏ï‡∏£‡∏á ‡πÉ‡∏´‡πâ‡∏´‡∏¢‡∏∏‡∏î

  // 3. ‡∏ï‡∏£‡∏ß‡∏à‡∏™‡∏≠‡∏ö‡∏ß‡πà‡∏≤‡∏Ñ‡∏≠‡∏•‡∏±‡∏°‡∏ô‡πå ID ‡πÉ‡∏ô‡πÅ‡∏ñ‡∏ß‡∏ô‡∏±‡πâ‡∏ô‡∏ß‡πà‡∏≤‡∏á‡πÄ‡∏õ‡∏•‡πà‡∏≤‡∏´‡∏£‡∏∑‡∏≠‡πÑ‡∏°‡πà
  const idCell = sheet.getRange(row, 2);
  const existingId = idCell.getValue().toString().trim(); 

  if (existingId === "") {
    try {
      const newId = generateUniqueCustomerId();
      idCell.setValue(newId);

    } catch (error) {
      Logger.log(`[ERROR] ‡πÑ‡∏°‡πà‡∏™‡∏≤‡∏°‡∏≤‡∏£‡∏ñ‡∏™‡∏£‡πâ‡∏≤‡∏á‡∏´‡∏£‡∏∑‡∏≠‡∏ï‡∏±‡πâ‡∏á‡∏Ñ‡πà‡∏≤ ID ‡πÅ‡∏ñ‡∏ß‡∏ó‡∏µ‡πà ${row}: ${error.toString()}`);
    }
  }
}

/**
 * ‡∏ü‡∏±‡∏á‡∏Å‡πå‡∏ä‡∏±‡∏ô‡∏ä‡πà‡∏ß‡∏¢‡πÉ‡∏ô‡∏Å‡∏≤‡∏£‡∏™‡∏£‡πâ‡∏≤‡∏á ID ‡∏•‡∏π‡∏Å‡∏Ñ‡πâ‡∏≤‡∏ó‡∏µ‡πà‡πÑ‡∏°‡πà‡∏ã‡πâ‡∏≥‡∏Å‡∏±‡∏ô
 * ‡∏£‡∏π‡∏õ‡πÅ‡∏ö‡∏ö: CUST-YYMMDD-HHMMSS-RANDOM_HEX
 * @returns {string} ID ‡∏•‡∏π‡∏Å‡∏Ñ‡πâ‡∏≤‡∏ó‡∏µ‡πà‡πÑ‡∏°‡πà‡∏ã‡πâ‡∏≥‡∏Å‡∏±‡∏ô
 */
function generateUniqueCustomerId() {
  const now = new Date();

  // ‡∏™‡πà‡∏ß‡∏ô‡∏Ç‡∏≠‡∏á Timestamp: YYMMDD-HHMMSS
  const YY = now.getFullYear().toString().slice(-2); 
  const MM = ('0' + (now.getMonth() + 1)).slice(-2); 
  const DD = ('0' + now.getDate()).slice(-2); 
  const HH = ('0' + now.getHours()).slice(-2); 
  const MI = ('0' + now.getMinutes()).slice(-2); 
  const SS = ('0' + now.getSeconds()).slice(-2); 

  const timestampPart = `${YY}${MM}${DD}-${HH}${MI}${SS}`;

  // ‡∏™‡πà‡∏ß‡∏ô‡∏Ç‡∏≠‡∏á Random Hex String 4 ‡∏´‡∏•‡∏±‡∏Å
  const randomHexPart = Math.floor(Math.random() * 0x10000).toString(16).toUpperCase().padStart(4, '0');

  // ‡∏õ‡∏£‡∏∞‡∏Å‡∏≠‡∏ö ID ‡∏ó‡∏±‡πâ‡∏á‡∏´‡∏°‡∏î
  return `CUST-${timestampPart}-${randomHexPart}`;
}
