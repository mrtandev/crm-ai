หลักสูตร AI Automation & Data Integration Master

โจทย์: การสร้างระบบจัดการลูกค้าสัมพันธ์อัตโนมัติ (Mini CRM Automation)

โจทย์นี้จะประเมินความสามารถของผู้เรียนในการประยุกต์ใช้เครื่องมือ No-Code (AppSheet) และ Workflow Automation (N8N) เพื่อสร้างโซลูชันที่สามารถจัดการข้อมูลลูกค้าและส่งการแจ้งเตือนอัตโนมัติ โดยเน้นความถูกต้องของ Logic Flow และการบูรณาการข้อมูล (Data Integration) ระหว่างแพลตฟอร์ม

สิ่งที่ต้องส่งมอบ (Deliverables)

  • AppSheet Application ที่สามารถเพิ่ม/แก้ไข/ดูข้อมูลลูกค้าได้
  • N8N Workflow ที่ตรวจสอบ Google Sheet ทุก 1 ชั่วโมง
  • ระบบส่ง Line Notify หรือ Email แจ้งเตือนไปยังเซลล์ เมื่อสถานะลูกค้าเปลี่ยนเป็น "สนใจสูง"
  • ผังงาน (Flowchart) แสดง Logic การทำงานของ Workflow

เกณฑ์การประเมิน (Rubrics)

  • Logic Flow (40%): ความถูกต้องและประสิทธิภาพของการออกแบบ Workflow (N8N)
  • Data Structure & App UI (30%): การออกแบบฐานข้อมูล (Google Sheet) และหน้าตาแอปพลิเคชัน (AppSheet) ที่ใช้งานง่าย
  • Integration (30%): ความสามารถในการเชื่อมต่อและส่งข้อมูลระหว่าง AppSheet และ N8N ได้อย่างราบรื่น

*ใบงานนี้เป็นตัวอย่าง หัวข้อจริงจะปรับเปลี่ยนตามความต้องการของพันธมิตร SMEs และระดับความยากจะเพิ่มขึ้นตามความคืบหน้าของหลักสูตร
