คู่มือการติดตั้ง AppSheet

Traceability Vinegar Database: จาก Schema สู่ App ใช้งานจริง

คู่มือนี้สรุปขั้นตอนที่ชัดเจนในการสร้างแอปพลิเคชัน AppSheet จากโครงสร้างฐานข้อมูล (`Structure`) ที่เราได้ออกแบบไว้ โดยเน้นการตั้งค่าความสัมพันธ์ (`Ref`) และ Logic ที่ซับซ้อนในตาราง `PRODUCTION_LINKS`

**เป้าหมาย:** สร้างแอปพลิเคชันที่สามารถตรวจสอบย้อนกลับข้อมูล Lot ได้ครบทั้ง 12 ตารางหลัก และเชื่อมโยงข้อมูลผู้ใช้

ขั้นตอนการ Setup AppSheet (5 ขั้นตอนหลัก)

1

การเตรียมข้อมูลสุดท้าย

  • **สร้างชีต `USERS`:** สร้างชีตชื่อ `USERS` ใน Google Sheets โดยมีคอลัมน์ `User_ID`, `Email`, `Full_Name`, `User_Role` และกรอกอีเมลผู้ใช้งานระบบอย่างน้อย 1 แถว
  • **ตรวจสอบ Header:** ยืนยันว่าชีตข้อมูลทั้งหมดมี **Header Row** เป็นภาษาอังกฤษตามที่ระบุในชีต `Structure`
2

นำเข้าตาราง & ตั้งค่าพื้นฐาน

  • **สร้างแอป:** เริ่มต้นสร้างแอปใหม่จาก Google Sheet `Traceability_Vinegar_database`
  • **นำเข้าตาราง:** ไปที่ **Data > Tables** และเพิ่มชีตที่เหลือทั้งหมด (12 Data Tables + `USERS` + `Structure`)
  • **Authentication:** ไปที่ **Security > Authentication** และตั้งค่าให้ใช้ **Google** เป็น Provider (Recommended)
  • **Audit Fields:** AppSheet ควรตั้งค่า `Created_By` และ `Modified_By` เป็น `Ref` ไปที่ตาราง `USERS` อัตโนมัติ
3

ตรวจสอบความสัมพันธ์ (Ref)

  • ตรวจสอบว่าคอลัมน์ที่มี `Type = Ref` ในชีต `Structure` ถูกตั้งค่าใน AppSheet ถูกต้อง
  • **ตัวอย่าง:** ในตาราง `FRUITS`, คอลัมน์ `Farm_ID` ต้องเป็น `Type: Ref` $\rightarrow$ `FARMS`
  • **App Formula:** ตรวจสอบว่า `Total_Weight_KG` และ `Modified_By/Date` ใช้สูตรตามที่กำหนดในชีต `Structure`
4

ควบคุม Form `PRODUCTION_LINKS`

ต้องใช้สูตร **Show If** เพื่อให้ผู้ใช้ป้อน Lot ID เพียงช่องเดียวในกลุ่ม **From** และ **To**

Logic ตัวอย่างสำหรับกลุ่ม From:

// คอลัมน์ From_Wine_ID จะแสดงก็ต่อเมื่อ Lot ก่อนหน้ายังไม่มีค่า
AND(ISBLANK([From_Raw_Lot_ID]), ISBLANK([From_Puree_Lot_ID]), ...)

**ทำซ้ำ** Logic นี้สำหรับคอลัมน์ `From_..._ID` และ `To_..._ID` ทั้ง 12 คอลัมน์

5

การออกแบบ UX & Report

  • **Traceability Dashboard:** สร้าง **Dashboard View** ที่รวม Detail View ของ `PACKAGINGS` และตาราง `Related...` Lot ต่างๆ เข้าไว้ด้วยกัน
  • **QR Scanning:** สร้าง **Action** ที่ใช้สูตร `LINKTOFILTEREDVIEW()` เพื่อสแกน QR Code แล้วนำค่า Lot ID ไปค้นหาใน Detail View ของตารางที่ถูกต้อง
