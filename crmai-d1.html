ชั่วโมงที่ 1: การเตรียมโครงสร้างพื้นฐาน (1 ชั่วโมง)

ชั่วโมงแรกคือการสร้างรากฐานของโปรเจกต์ CRM AI โดยการติดตั้งเครื่องมือหลัก 3 ตัว ได้แก่ n8n, GitHub และ Gemini API

1.1 สมัคร n8n Pro และติดตั้งเบื้องต้น (20 นาที)

n8n คือ **"ท่อประปาอัจฉริยะ"** ที่จะเชื่อม AI และระบบทั้งหมดเข้าด้วยกัน การเลือกแผน Pro ให้ความยืดหยุ่นที่เราต้องการ

ขั้นตอนการปฏิบัติ:

  1. ไปที่ Pricing ของ n8n และเลือกแผน **Pro** เพื่อให้ได้ Executions 10,000 ครั้ง/เดือน
  2. เลือกชำระแบบ **Annual** เพื่อรับส่วนลด **20%** (ประหยัดได้จาก €50/เดือน เหลือประมาณ €40/เดือน หรือประมาณ $510 ต่อปี)
  3. ดำเนินการชำระเงินและตั้งค่าบัญชี Cloud Hosting ของ n8n

1.2 สร้าง GitHub Repo (Free) สำหรับโปรเจกต์ CRM (20 นาที)

GitHub จะเป็นที่เก็บโค้ด (ถ้ามีการใช้ Node Code ใน n8n) และใช้เป็นเครื่องมือจัดการเวอร์ชันของโปรเจกต์

ขั้นตอนการปฏิบัติ:

  1. ไปที่เว็บไซต์ GitHub และสมัคร/เข้าสู่ระบบบัญชีของคุณ
  2. สร้าง Repository ใหม่ (New Repository) ตั้งชื่อว่า `crm-ai-sme`
  3. เลือก **Private** (เพื่อเก็บโค้ดเป็นความลับ) และเลือก License เป็น `MIT`
  4. ทำการ Clone Repo นี้ลงในเครื่องคอมพิวเตอร์ของคุณเพื่อเตรียมทำงานต่อไป

1.3 เชื่อมต่อ Gemini API และทดสอบการเรียกใช้ (20 นาที)

เราจะตั้งค่า Gemini Pro เพื่อให้ n8n สามารถเรียกใช้พลัง AI สำหรับ Lead Scoring และ Document Generation ได้

ขั้นตอนการปฏิบัติ:

  1. ไปที่ Google AI Studio หรือ Console เพื่อสร้าง **API Key** สำหรับ Gemini (ห้ามเปิดเผยคีย์นี้!)
  2. ใน n8n Dashboard ให้ไปที่ **Credentials** แล้วสร้าง Credential ใหม่สำหรับ **Google Gemini**
  3. ใส่ API Key ที่ได้มา และตั้งชื่อ Credential ว่า `Gemini Pro CRM`
  4. **ทดสอบ Node:** สร้าง Workflow ชั่วคราว เพิ่ม Node Gemini และลอง Prompt ง่ายๆ เช่น "Summarize the benefits of AI for SMEs" เพื่อยืนยันว่าเชื่อมต่อสำเร็จ
