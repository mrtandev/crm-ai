คู่มือปฏิบัติการ: CRM AI Automation สำหรับ SMEs

สารบัญหลัก

ชั่วโมงที่ 5: Testing, Debugging และกลยุทธ์ Scale (1 ชั่วโมง)

ชั่วโมงสุดท้ายคือการทำให้ระบบมั่นคง! เราจะทดสอบการทำงานตั้งแต่ต้นจนจบ (End-to-End) และวางแผนการลงทุนเพื่อเติบโตในอนาคต เพื่อเพิ่ม **BH (Better Health)** ของระบบในระยะยาว

5.1 ทดสอบระบบ End-to-End (20 นาที)

เราต้องจำลองสถานการณ์จริงเพื่อยืนยันว่า Workflow ทำงานถูกต้องในทุกขั้นตอน โดยไม่เกิดข้อผิดพลาดจาก Gemini หรือการเชื่อมต่อ

ขั้นตอนการปฏิบัติ:

  1. **Test Case 1 (Hot Lead):** ส่งข้อมูล Lead ที่มีคุณภาพสูง (ตาม Prompt ของ Gemini) แล้วตรวจสอบว่า:
    • Google Sheet มีการบันทึก Score สูง (>7) และ `is_hot` เป็น `true`
    • ลูกค้าได้รับอีเมลใบเสนอราคาจากชั่วโมงที่ 4
  2. **Test Case 2 (Cold Lead):** ส่งข้อมูล Lead ที่ไม่น่าสนใจ แล้วตรวจสอบว่า:
    • Google Sheet มีการบันทึก Score ต่ำ และ `is_hot` เป็น `false`
    • Workflow หยุดทำงานที่ IF Node (ไม่ส่งอีเมลเสนอราคา)
  3. **Test Case 3 (Error):** ทดสอบส่งข้อมูลที่ไม่ครบถ้วน (เช่น ไม่มีอีเมล) เพื่อดูว่า n8n จัดการ Error ได้อย่างไร

5.2 Debugging และการตรวจสอบ Log (20 นาที)

ใช้ฟีเจอร์ของ n8n Pro เพื่อแก้ไขปัญหาที่เกิดขึ้น (เช่น โค้ดที่ล้มเหลว หรือ API error)

ขั้นตอนการปฏิบัติ:

  1. **ใช้ Execution Search:** ใน n8n ให้ไปที่ **Executions** แล้วค้นหา Workflow ที่ล้มเหลว
  2. **วิเคราะห์ Log:** ดูว่า Node ไหนที่ทำงานผิดพลาด แล้วตรวจสอบ `Output` ของ Node ก่อนหน้าเพื่อหาต้นเหตุ (เช่น Gemini ส่ง JSON ผิดรูปแบบ)
  3. **ปรับปรุง Prompt:** หาก AI ให้คะแนนผิด ให้กลับไปปรับปรุง Prompt ใน Gemini Node ให้ชัดเจนยิ่งขึ้น (Prompt Engineering)
  4. **บันทึก Code ใน GitHub:** หากมีการแก้ไขโค้ดใน Node Code ให้ทำการ Commit และ Push ไปยัง Repo ใน GitHub ที่สร้างไว้ในชั่วโมงที่ 1

หน้าจอจำลอง: การตรวจสอบ Execution Log

ERROR: Google Sheets Node Failed

Input data structure mismatch (JSON keys not found).

**วิธีการแก้:** ตรวจสอบ Output ของ Gemini Node ว่า JSON แตกต่างจากที่คาดหวังหรือไม่

5.3 กลยุทธ์การ Scale และอัปเกรด (20 นาที)

เมื่อระบบ CRM AI เริ่มสร้างรายได้ คุณควรวางแผนอัปเกรดเครื่องมือเพื่อเพิ่มความเร็วในการพัฒนา

แผนการอัปเกรด:

  • **Upgrade 1: GitHub Copilot Pro ($120/ปี):** เมื่อเริ่มมีการเขียนโค้ดซับซ้อนใน Node Code เป็นประจำ นี่คือการลงทุนที่ให้ผลตอบแทนคุ้มค่าที่สุด (**ST**, **MM**)
  • **Upgrade 2: Gemini AI Ultra:** อัปเกรดเมื่อต้องทำ AI Analysis ที่ซับซ้อนมาก เช่น การวิเคราะห์เอกสารทางกฎหมายหรือข้อมูลทางการเงินปริมาณมหาศาล
  • **Upgrade 3: n8n Enterprise:** เมื่อมีผู้ใช้งานในทีมมากกว่า 5-10 คน และต้องการฟีเจอร์ด้านความปลอดภัยขั้นสูง (SSO/Git Control)

คุณได้สร้างระบบ CRM AI Automation ที่มั่นคงและพร้อมสำหรับการเติบโตในโลกธุรกิจ SMEs แล้ว
