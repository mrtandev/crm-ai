ชั่วโมงที่ 4: Sales & Document Automation (1 ชั่วโมง)

ชั่วโมงนี้เราจะเปลี่ยน **Lead คุณภาพสูง** ให้กลายเป็นการปิดการขาย โดยใช้ AI ร่างเอกสารและส่งต่อให้ลูกค้าโดยอัตโนมัติ เพื่อเพิ่ม **ST (Save Time)** ให้กับทีมเซลส์

4.1 กรอง Lead คุณภาพสูง (Hot Lead) (15 นาที)

เราจะใช้คะแนน `is_hot` ที่ได้จาก Gemini Pro ในชั่วโมงที่ 2-3 มาเป็นเงื่อนไขในการทำงานต่อ

ขั้นตอนการปฏิบัติ:

  1. เพิ่ม Node ชื่อ **"IF"** ต่อจาก Node ที่บันทึก Google Sheets (จากชั่วโมงที่ 2-3)
  2. ตั้งค่าเงื่อนไข: **If** `is_hot` **is equal to** `true` ให้ทำงานใน Branch True (Hot Lead)
  3. Branch False (Cold Lead) อาจส่งไป Workflow ติดตามผลอัตโนมัติ (Assignment: ลองทำเอง)

หน้าจอจำลอง: IF Node (Filtering)

Condition:

{{ $json.is_hot }} == true

✅ Branch True (Hot Lead) ❌ Branch False (Cold Lead)

4.2 ใช้ Gemini Pro ร่างใบเสนอราคาเบื้องต้น (25 นาที)

เราจะส่งข้อมูล Lead คุณภาพสูงกลับไปให้ AI เพื่อร่างเอกสารที่ปรับเนื้อหาให้เข้ากับลูกค้าแต่ละรายทันที

ขั้นตอนการปฏิบัติ:

  1. เพิ่ม Node **"Google Gemini"** ต่อจาก Branch True ของ IF Node
  2. สร้าง **Prompt** สำหรับร่างใบเสนอราคา โดยอ้างอิงข้อมูลลูกค้าจาก Webhook

หน้าจอจำลอง: Prompt ร่างใบเสนอราคา

Prompt (สั่งให้ AI เป็นนักเขียน):

Act as a professional Sales Manager. Draft a preliminary quotation for {{ $json.name }}'s need for "{{ $json.interest }}". Include a suggested price range (e.g., 80,000 - 120,000 THB) and 3 key benefits. Write in formal Thai.

4.3 ส่งใบเสนอราคาทางอีเมล (20 นาที)

เราจะใช้ Node สำหรับส่งอีเมลเพื่อส่งเนื้อหาที่ Gemini ร่างไว้ไปยังลูกค้าโดยอัตโนมัติ

ขั้นตอนการปฏิบัติ:

  1. เพิ่ม Node ชื่อ **"Gmail"** หรือ **"Send Email"** อื่นๆ (เช่น Outlook, SendGrid)
  2. ตั้งค่า **Recipient** เป็นอีเมลของลูกค้า (`{{ $json.email }}`)
  3. นำข้อความที่ได้จาก Gemini มาใส่ในช่อง **Body** ของอีเมล
  4. (Assignment) เพิ่ม Node **"Google Drive"** เพื่อบันทึกไฟล์ PDF/เอกสารที่สร้างขึ้น

หน้าจอจำลอง: Gmail Node (Send Email)

To:

{{ $json.email }}

Subject:

ใบเสนอราคาเบื้องต้นสำหรับคุณ {{ $json.name }}

Body:

{{ $json.gemini_output.text }}

สรุปท้ายชั่วโมงที่ 4: การปิดการขายอัตโนมัติเสร็จสมบูรณ์!

คุณได้สร้าง Workflow ที่จัดการ Lead, กรองคุณภาพ, ร่างใบเสนอราคา, และส่งอีเมลออกไปหาลูกค้าโดยไม่ต้องมีคนเข้ามาแทรกแซง
